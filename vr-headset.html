physical product design

introduction

Group Members:

Donny Wu, Ridge Zhang, Andrea Ma.

This final project was made for the Material in Design course, IAT 336, A course focused on how materials behave, manufacturing techniques, and physical product design.

Working in a group of 3, we designed, prototyped, and create a VR headset within 2 months

The constraint was we had to use plastic and wood and being mindful of ergonomics, ease of use, and a specific persona.

tools used

Sketching

  • Used for initial design and iteration

Autodesk Maya

  • Low poly 3d modelling
  • Used for secondary refinements and prototyping using laser cutters utilizing UV unfold and packing features
  • Also used to form the VR headset to a human head for a comfortable fit.

SolidWorks

  • Final 3D printing STL creation, here I added many alignment tabs, precise slot creation for hinges and lenses, and foam cutting pattern creation.

design context

User persona: stargazer

  • Can be used outside
  • Easily remove and put on
  • Connected with nature
  • Portable enough to take around while on hikes

Design constraint

One of the constraints we had while designing this VR headset is that we were constrained to using two different materials, wood and plastic, at roughly a 50/50 ratio on the exterior. This restriction makes it not feasible to create compound curves. When I was sketching, I kept the tools available to us in mind, which is a laser cutter for wood and a 3d printer for plastic.

first iteration

In the first iteration, a challenge I had was how to convert a 3D model from Maya to a laser cut file. I thought about UV unwrapping it but the UV map exporter only exports in raster formats. After a couple hours of research, I've found a script that can set each vertex's coordinates to UV coordinates, creating a flat plane model in the 3d view, which that can be rendered by the Maya's vector renderer. The result was an illustrator file that needed very little editing to get into a laser cutter readable format.

second iteration

The second iteration model was made to see how it fits around the face. It came out really well. This first attempt at form fitting the face went well because I used a 90th percentile sized 3D scanned head to make my contours around the face.

The model was made out of cardstock since it is easy to laser cut, folds right on the dashed lines, and won't buckle at non-perforated lines. This helps in the folding and gluing process. A problem we faced was accounting in for the paper thickness because we did not know what kind of cardstock we were going to use to cut. So, when assembling, we needed to put a lot of effort into stretching and glueing for the top part to complete the shell to close.

final iteration

In the second iteration, everything went really well. I was really proud of my work making the 3D printed STL file and laser cut file.

The 3D print STL file was created in SolidWorks with a single model split into 4, each with added support for hinges, alignment slots, and draft angles for a nice fit while glueing.

The Laser cut file was created using the UV unwrapped Maya geometry with manual scaling to match the size of the real size model. Each face of the unwrapped model was mapped with a flexible wood cutting pattern. This enables us to curve the wood in compound curves.

reflection

During the course of this project, I've applied and extended a lot of what I have learned in the School of Interactive Arts and Technology. With solid works, I have learned a full array of new features include boolean operations, derived parts, importing files from Maya and lofting. I have learnt what software is good for what task.

At the end at the final public showcase, many people were impressed with the final project and it fits perfectly around the professor's head. We received the highest mark in the class for this project.
